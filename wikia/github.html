GitHub

Tutorial de cómo actualizar tu web sin necesidad de WebHooks

Imagen representativa

En éste tutorial mostraremos cómo actualizar tu web mediante script.

Paso 1

Desde una terminal escribiremos el comando apt-get update para actualizar los paquetes.

Una vez actualizados los paquetes utilizaremos el comando apt-get install git.

Paso 2

Configuraremos los parámetros iniciales

git config --global user.name "nombreusuariogithub"

git config --global user.email "emaildecuentagit"

Paso 3

Creamos una carpeta llamada scripts en nuestro directorio /home con el comando
mkdir -p /home/scripts

Dentro de la carpeta, creamos el archivo con nano /home/scripts/git.sh

#Inicio del script
#!/bin/bash
cd rutadelrepositorioweb
#Ejecutamos los comandos de actualización
git fetch origin master
git merge origin master
# Finalizamo el script
exit

Paso 4

Ahora añadiremos el archivo script al sistema crontab

Usamos el comando crontab -e, al final del archivo añadimos:

*/1 * * * * /home/scripts/git.sh

Guardamos y salimos
