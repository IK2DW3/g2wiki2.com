Apache2

Comienzo del tutorial del servicio web

Imagen de muestra montrando el resultado del comando: systemctl status apache2

En éste tutorial mostraremos cómo hacer un servidor web sencillo con el recurso Apache2.

Paso 1

Desde una terminal escribiremos el comando apt-get update para actualizar los paquetes.

Una vez actualizados los paquetes, proseguiremos a usar el comando apt-get install apache2.

Paso 2

Crearemos nuestro propio sitio avilitado a partir del sitio por defecto que se genera en la instalación, ese archivo se encuentra en el directorio /etc/apache2/sites-enabled con el nombre de
000-default.conf.

Creamos una copia del archivo con el comando cp /etc/apache2/sites-enabled/000-default.conf /etc/apache2/sites-enabled/tusitio.com.conf

Paso 3

Imagen de muestra del comando: nano /etc/apache2/sites-enabled/tusitio.com.conf

ServerName ejemplo.com
ServerAlias www.ejemplo.com
ServerAdmin correo@ejemplo.com

DocumentRoot /var/www/ejemplo.com
DirectoryIndex index.html o index.php

  • ServerName: nombre del servidor de resolucion de nombres (DNS)
  • ServerAlias: alias que se le asigna a la máquina o nodo.
  • ServerAdmin: servidor de correo
  • DocumentRoot: directorio raiz del sitio web
  • DirectoryIndex: archivo principal que carga por defecto el sitio web

Los parámetros ErrorLog y CustomLog son opcionales.

Paso 4

Una vez configurado nuestro archivo de configuración del sitio web, usamos el comando
apache2ctl configtest para la confirmación de errores.

Paso 5

Para finalizar, reiniciamos el servicio web con el comando /etc/init.d/apache2 restart.

Usando un navegador web, accedemos a http://localhost:80/ o http://localhost:8088/
