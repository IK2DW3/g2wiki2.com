ProFTPD

Comienzo del tutorial del servicio comparticion archivos en nube

Imagen de muestra montrando el resultado del comando: systemctl status proftpd

En éste tutorial mostraremos cómo hacer un servidor de comparticion de archivos con el recurso ProFTPD.

Paso 1

Desde una terminal escribiremos el comando apt-get update para actualizar los paquetes.

Despues escribiremos apt-get install proftpd para instalar el servicio.

Paso 2

Creamos un usuario con el cual usaremos para iniciar sesión en FTP mediante web o la aplicación FileZilla.

Usamos la siguiente instrucción:
adduser < usuario > -shell /bin/false -home "< direccion-carpeta >"

Nos pedirá que rellenemos al información sobre el nuevo usuario, lo mas importante es poner la contraseña y confirmarla.

Paso 3

Ahora configuraremos el archivo proftpd.conf que se encuentra en /etc/proftpd/ usando la instrucción nano /etc/proftpd/proftpd.conf

Añadiremos las siguientes lineas al final del archivo:
< Global >
RootLogin off
RequireValidShell off
< /Global >

Guardamos, cerramos el archivo y reiniciamos el servicio: service proftpd restart

Paso 4

Cambiar los permisos del directorio para el uso del usuario: chown -R < usuario > < direccion >
